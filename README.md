# ShopForge AI

ShopForge AI is a premium Shopify Headless Commerce Platform built for scalability, performance, and modern user experiences.

## Tech Stack

- **Frontend:** Next.js (App Router), React, TypeScript, Tailwind CSS
- **Commerce:** Shopify Storefront API, Shopify GraphQL
- **Deployment:** Vercel
- **AI Integration:** OpenAI (Product Description, SEO, Search, Recommendations)

## Core Principles

- **Clean Architecture:** Separation of concerns and modular design.
- **Feature-based Structure:** Organized by domain logic for maintainability.
- **Performance First:** Optimized Core Web Vitals and server-side rendering where appropriate.
- **Premium Design:** Minimal, elegant, and responsive UI inspired by Apple, Stripe, and Linear.

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- Shopify Store & Storefront API Access Token

### Installation

1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd shopforge-ai
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env.local` file based on `.env.example`.

4. Run the development server:
   ```bash
   npm run dev
   ```
